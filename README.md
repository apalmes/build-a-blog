# Blog App
## A simple Flask project for LC-101 students

_Implements_:

- [Flask](http://flask.pocoo.org/) template rendering
- manual form validation
- HTTP routing & endpoints
- HTML
- CSS
- request
- redirect
- render_template (in place of jinja2)
